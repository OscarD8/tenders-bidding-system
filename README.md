### Quick Start

1. The Python/Django environment will be automatically configured
2. Navigate to the Django Project folder and run:
   ```bash
   cd "Django Project"
   python manage.py runserver
   ```
3. Access the Django application at the forwarded port (typically port 8000)

### Dependencies

- Python 3.12
- Django 5.2.6 (latest version)
- All dependencies are listed in `requirements.txt`

The development environment is pre-configured with VS Code extensions for Python and Django development.
