Prompt: Change logic of hovering on a card due to creation of a new class for first items in a row of cards which had different functionality.
"class="card card--sentinel" This specific class does not have overlay functionality. Fix the code to stop hover triggering background to empty

Date: 17-10-25

Copilot applied the 

not()
psuedo-class which was new to me. This meant that the overlay of a white background linked to showing text when hovering over a card did not trigger for divs with the class .card--sentinel. This was centered around a card being represented by two divs - one for the overlay text with white background and one for just an image. On hover changed the opacity and display of each (as long as not a .card--sentinel which copilot named...). 
/* only show overlay for non-sentinel cards */
.card:not(.card--sentinel):hover .card__overlay,
.card:not(.card--sentinel):focus .card__overlay {
  opacity: 1;
  background: #fff;

/* do NOT hide the image for sentinel cards; only hide for normal cards */
.card:not(.card--sentinel):hover .card__image,
.card:not(.card--sentinel):focus .card__image {
  display: none;
}
}

Prompt: I need CSS to create a custom horizontal scrollbar for a div with the class .custom_scrollbar. It should use flexbox to align items in a row. The width should be 50vw. It must scroll horizontally but hide any vertical scrollbar. Make the background of the scrollbar red.

Date: 17-10-25

The response was useful in stating that the elements of the scrollbar should be displayed with flex as a row, then asserting the overflow-x as auto and overflow-y to be none. I ended up not using any of the other functionality it suggested with the webkit psuedo elements as it wasn't changing anything for me, which was confusing. I was also having a hard time figuring out how to colour it and eventually (as none of these psuedo elements were working) on MDN docs found the property for 'scrollbar-color' which could be applied directly to the div that wrapped the row of cards and this worked. 

.custom_scrollbar {
  display: flex;
  flex-direction: row;
  align-items: center;
  width: 50vw;
  overflow-x: auto;
  overflow-y: hidden;
  height: 2em;
  padding: 1em;
  margin: 1em auto;
  background-color: #333;
}

/* 1. Define the size of the entire scrollbar */
.custom_scrollbar::-webkit-scrollbar {
  height: 16px; /* Sets the height for the horizontal scrollbar */
}

/* 2. Style the track */
.custom_scrollbar::-webkit-scrollbar-track {
  background: red;
}

/* 3. Style the draggable thumb (now correctly scoped) */
.custom_scrollbar::-webkit-scrollbar-thumb {
  background: #888;
  border: 2px solid red; /* Added a border to make it pop against the red */
}

/* 4. Style the arrow buttons (now with a defined size) */
.custom_scrollbar::-webkit-scrollbar-button {
  background: red;
  display: block;
  width: 16px;
  height: 16px;
}
