Date: 03-10-2025

This was part of the week 2 lab exercise, using copilot to generate an html file with a linked stylesheet. The index.html that it generated split out the header, nav, main and footer elements as required. To achieve the required segments of 1:4:1 weighting in the main section - it implemented a div with class `main-row`, inside which it placed two aside elements with `col right` and `col left` classes, as well as a section with class `col center`. This structure makes the CSS easy to understand. The `main-row` container has flex applied and all nested elements are referred to by class and their corresponding flexible length is applied: 

/* CSS generated for main content flex application */

.main-row {
  display: flex;
  gap: var(--gap);
  align-items: stretch;
  width: 100%;
  max-width: 1100px;
  margin: 0 auto;
}

.main-row > .col {
  padding: 1rem;
  border-radius: 10px;
  color: var(--text);
}
.left {
  flex: 1;
  background: linear-gradient(180deg, #062a36, #08303b);
  border: 1px solid rgba(255, 255, 255, 0.02);
}
.center {
  flex: 4;
  background: linear-gradient(180deg, #041926, #06232c);
  border: 1px solid rgba(255, 255, 255, 0.02);
}
.right {
  flex: 1;
  background: linear-gradient(180deg, #062a36, #08303b);
  border: 1px solid rgba(255, 255, 255, 0.02);
}

Date: 26-09-2025

The original style sheet didn't cascade the style optimally. Codepilot was used to make a suggestion and refactor the CSS. As an example: 

/* Original CSS */
#html .card {
  background-color:#ebfffc ;
  border-radius: 4px;
  border: 1px solid #dbdbdb;
  color: #363636;
  font-size: 1rem;
  margin-bottom: 1.5rem;
  position: relative;
}
/* Shared card styles */
.card {
    border-radius: 4px;
    border: 1px solid #dbdbdb;
    color: #363636;
    font-size: 1rem;
    margin-bottom: 1.5rem;
    position: relative;
}
/* Refactored CSS suggested by AI */
#html .card { background-color: #ebfffc; }

The refactor also introduced "composes: page-title;" which is a conceptual way to show reuse. 

.page-title {
    font-family: "Montserrat", sans-serif;
    font-weight: 700;
    border-bottom: #222525 2px solid;
    font-size: 2.5em;
    color: #333;
    margin-bottom: -0.3em;
}

#html > h1 { composes: page-title; }
